## 🔧 Common Issues & Fixes

**❌ Installation fails with "command not found"**
- Ensure you've copied the entire command correctly from the GitHub repository
- Make sure you have internet connectivity during installation
- Try running the command without any modifications

**❌ Permission denied errors**
- You need to enter your admin password when prompted in Terminal
- Ensure you have administrator privileges on your Mac
- Check System Settings > Privacy & Security for any blocked processes

**❌ "Developer cannot be verified" warning**
- This is a standard macOS security message for unsigned applications
- Go to System Settings > Privacy & Security and click "Open Anyway"
- Alternatively, hold the Option key while clicking "Open" from the context menu

**❌ CleanMyMac doesn't detect any issues**
- This might happen if your system is already clean — try running a deep scan
- Ensure the tool has necessary permissions in System Settings > Privacy
- Check that you're using the latest version with current definitions

**❌ Slow scanning speed**
- Close other applications during the scan
- Ensure you have at least 1GB of free RAM
- The first scan may take longer; subsequent scans are faster

**❌ macOS update breaks compatibility**
- This is expected behavior — download the latest release from the repository
- The team typically pushes updates within 24-48 hours of major macOS releases
